  1️⃣  Start the backend:
      cd MyWeb && python app.py
      → Look for: ✅ MQTT Connected, 🚀 Server starting

  2️⃣  Start the frontend:
      cd smart-farm-dashboard && npm run dev
      → Open: http://localhost:5173

  3️⃣  Check status:
      → Dashboard should show: 🟢 ONLINE
      → Browser console should show: ✅ Connected to Server!

  4️⃣  Send test data:
      mosquitto_pub -h localhost -t "smartfarm/sensors" -m '{
        "air": {"temp": 25.5, "hum": 65},
        "soil": {"hum": 80, "ph": 6.8, "n": 150, "p": 100, "k": 120},
        "env": {"lux": 1000, "co2": 450}
      }'

  5️⃣  Verify:
      → Values update on dashboard
      → Graph shows new data point
      → Database saved the record
