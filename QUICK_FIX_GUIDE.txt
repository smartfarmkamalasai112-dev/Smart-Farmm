🆘 IF YOU GET ERRORS
═══════════════════════════════════════════════════════════════════

ERROR: "Port 5000 already in use"
FIX:   pkill -9 python
       python app.py

ERROR: "Port 5173 already in use"
FIX:   pkill -9 node
       npm run dev

ERROR: "Connection refused"
FIX:   1. Make sure backend is running (Terminal 1)
       2. Check IP in smart-farm-dashboard/src/App.jsx line 15
          const SOCKET_URL = "http://YOUR_IP:5000";
       3. Restart frontend (npm run dev)

ERROR: "Cannot find module"
FIX:   cd smart-farm-dashboard
       npm install
       npm run dev

ERROR: "MQTT Connection failed"
FIX:   1. Check broker running: sudo systemctl status mosquitto
       2. Verify IP in MyWeb/app.py line 20:
          MQTT_BROKER = "localhost"
